# SQL_DE25

Repository for the course SQL at STI - Data Engineer. This repo is used for code along in the lectures, videos and to preform exercises and more.

## Folder structure

Each part of this course has it's own main-folder.

- code_along_lectures
- code_along videos
- Exercises

### code_along_videos

This folder contaings the material for the videos from the course-repo: https://github.com/AIgineerAB/duckdb_sql_analytics_course.git

The code inside is from coding along the videos.

Inside each chapter/video there is folders and material for:

- SQL-files
- duckdb database
- ui
- data

### code_along_lectures

The material in this folder are from the physical and online lectures in this course.

Each folder has the name of the lecure date (lecture_date)

The structure in these are the same as the code_along_videos:

- SQL-files
- duckdb database
- ui
- data

### Exercises

This folder contains subfolders with the exercises from this course.
